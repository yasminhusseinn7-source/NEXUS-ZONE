# 🎮 NEXUS ZONE - Professional Gaming Center Landing Page

**NEXUS ZONE** is a modern, high-end landing page designed for a premium gaming lounge. It features a **Cyberpunk/Futuristic** aesthetic with neon accents, designed to provide a high-energy atmosphere for gamers.

---

## ✨ Key Features
* **Futuristic UI/UX:** A dark-themed interface with Cyan and Magenta neon highlights.
* **Responsive Grid System:** Perfectly aligned "Features" and "Pricing" cards that adapt to all screen sizes.
* **Interactive Elements:** * Custom-made floating cursor.
    * Glitch animations on the logo.
    * Scroll-reveal animations for a dynamic browsing experience.
* **Booking System:** A sleek, user-friendly reservation form.
* **Performance Focused:** Clean code structure using CSS Flexbox and Grid for maximum speed and stability.

---

## 🛠️ Built With (Tech Stack)
* **HTML5:** Structured for SEO and accessibility.
* **CSS3:** Utilizing CSS Variables, Flexbox, Grid, and Custom Keyframe Animations.
* **JavaScript (Vanilla):** For smooth scroll reveals and the interactive custom cursor.
* **Google Fonts:** `Cairo` (Arabic/English support) and `Rajdhani` (for that sharp, techy look).
